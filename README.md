# 🌦️ Weather App

A simple and clean Flutter weather application that fetches real-time weather information using the OpenWeather API.

## 📱 Features

- 🔍 Search weather by city name
- 🌡️ Display current temperature
- 🏙️ Show city name
- ☁️ Display weather description
- 💧 Show humidity
- 🌬️ Show wind speed
- 🌤️ Display weather icon from the API
- ⏳ Loading indicator while fetching data
- ❌ Error handling using SnackBar
- 📱 Clean and responsive UI

## 🚀 Getting Started


